# PHC Analytics — Mock (KPIs & Dashboards)

## Objetivo
Sistema de análise de dados empresariais baseado em dados PHC (mock),
com KPIs estratégicos e dashboards interativos.

## Estrutura
- 01_Notebooks: geração de dados mock e cálculo de KPIs
- 02_Data_Mock: modelo estrela + outputs de KPIs
- 03_Dashboard: aplicação Streamlit
- 04_Documentacao: notas técnicas

## KPIs Implementados
- Faturação mensal
- Margem mensal (%)
- Ticket médio
- Top clientes
- Clientes inativos (>180 dias)
- Top artigos por faturação e margem

## Dashboard
Aplicação Streamlit com 3 vistas:
- Visão Geral
- Clientes
- Produtos / Serviços

Pronto para ligação a dados reais PHC quando houver acesso à BD.
